# main-portal-lab

## 项目简介

`main-portal-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态页面资源的集合与存储空间。每个 HTML 文件都可以独立运行，适用于快速原型、小型工具页面或演示内容的托管。

## 目录说明

```
main-portal-lab/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 公共资源文件（CSS、JS、图片等）
├── docs/           # 相关文档或说明文件
└── README.md       # 本文件
```

- **pages/**：每个子文件夹或文件对应一个独立页面，命名清晰，便于识别。
- **assets/**：存放多个页面可能共享的静态资源，避免重复。
- **docs/**：用于存放与页面相关的设计文档、使用说明或版本记录。

## 页面归档说明

本仓库收录的 HTML 页面均以独立形式存在，彼此之间无依赖关系。每个页面都包含完整的 HTML 结构，可直接在浏览器中打开使用。归档内容可能包括：

- 工具类页面
- 信息展示页
- 实验性原型
- 其他静态 HTML 资源

所有页面不包含任何外部追踪、营销信息或指向特定网站的链接，仅提供基础功能与内容。

## 维护说明

- 本仓库由维护者定期更新，新增页面或修正已有内容。
- 提交新页面时，请放入 `pages/` 目录，并尽量保持命名规范。
- 如涉及公共资源更新，请同步修改 `assets/` 下的对应文件。
- 欢迎通过 Issues 提交建议或报告问题，但本仓库不接受直接的外部贡献者拉取请求。
